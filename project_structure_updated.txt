🆕 NEW CROSS-PLATFORM FEATURES:
✅ Windows & macOS Support
✅ Platform-specific keyboard shortcuts (Ctrl vs Cmd)
✅ Native app detection for both platforms
✅ Cross-platform GUI with platform indicators
✅ Automatic dependency management per platform
✅ Universal launcher with auto-detection
✅ Platform-specific setup scripts
✅ macOS accessibility permissions handling
✅ PyObjC integration for native macOS features
✅ Comprehensive documentation for both platforms

🎯 PLATFORM DIFFERENCES HANDLED:
• Windows: Uses Ctrl key, win32gui for window detection, pywin32 for system integration
• macOS: Uses Cmd key, AppKit/Cocoa for app detection, requires accessibility permissions
• Shortcuts: Automatically maps Windows Ctrl shortcuts to macOS Cmd shortcuts
• Process Detection: Different methods for each platform's process management
• GUI: Shows current platform in title bar and status

🚀 USAGE:
1. Windows: Run quick_start_windows.bat OR python3 start_server.py
2. macOS: Run ./quick_start_mac.sh OR python3 start_server.py
3. Universal: python3 start_server.py (auto-detects and runs appropriate setup)

📱 ANDROID APP:
• Same app works with both Windows and macOS servers
• No platform-specific changes needed
• Just enter the IP address of your Windows PC or Mac
• All TourBox-inspired controls work identically on both platforms
